Tips and hints

Document Object Model

This is the hierarchial layout of the HTML document:

!DOCTYPE html': tells you what version of html you're using
'html': this can be used when using html5
'head'
'title'Page title'/title'
'/head'
'body'
...
''/body'
''/html'

Key to remember

  1. Always start the html file writing "!DOCTYPE html"
    2. This will allow the browser to know which version of html I am using
  2. A coherent style is necessary; having a head, body and styles within

Common Tags

  • heading (h?): h1 for largest to h6 for smallest
  • paragraph (p): paragraph of a text
  • bold (b): text between tags in bold
  • italic (i): text between tags in italic
  • link ('a href="url"'' link name '/a'): creates a link to another page or website
  • image ('img src="filename.jpg"''): show an image
  • list ('ul''ol''li' list ''/li''ol''/ul'): unordered, ordered, bullet-point list
